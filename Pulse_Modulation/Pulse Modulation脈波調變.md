# 脈波調變的分類
### 類比脈波調變(Analog Pulse Modulation)
週期性的脈波烈被拿來當載波，為連續性的變化。</br>
1. PAM: Pulse Amplitude Modulation
2. PWM: Pulse Width Modulation
3. PPM: Pulse Position Modulation </br>
